Welcome to Online Pāḷi Tipiṭaka Website!

This website (website source code) allows instant lookup of words when mouse cursor hovers over words. The dictionaries include Pāḷi-English, Pāḷi-Japanes, and Pāḷi-Chinese. The data of Pāḷi Tipiṭaka in this website comes from Pāḷi Tipiṭaka. Before using this website, please read Howto on Pāḷi Tipiṭaka first. If you only need to look up Pāḷi words, please visit Pāḷi Dictionary (website source code). The data of dictionaries comes from Pali Dict Linux Web Ver 1.0. This website is still under development, if any questions, problems, or suggestions, please contact me.

Firefox and Chrome are preferred browsers for this website. IE8 or older are not supported. Users who use old browsers please visit old version of this website.
